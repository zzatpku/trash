初始化 git 仓库：git init.

将一个文件纳入跟踪/将一个文件的修改提交到缓冲区：git add <path/to/the/file/or/dir>

修改 git 提交的用户名和邮箱

```bash
git config --global user.name <name>
git config --global user.email <email>
```
